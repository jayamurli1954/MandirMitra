## Project Overview
MandirMitra is a comprehensive temple management system designed for religious institutions across India.

## Key Features
- Vedic calendar integration with Panchang support
- Donation tracking and automated receipt generation
- Religious event scheduling and ceremony management
- Multi-language support (Hindi, Sanskrit, regional languages)

## Technology Stack
- **Frontend:** React with culturally-appropriate UI components
- **Backend:** Python FastAPI with Panchang API integration
- **Database:** MongoDB with religious calendar data
- **Integrations:** Payment gateways, SMS services

## Integration with SanMitra Portfolio
- **MitraBooks:** Financial management for donations and expenses
- **LegalMitra:** Religious trust compliance and legal documentation
- **Shared Components:** Authentication, notifications, analytics

## Getting Started
1. Create documentation: `.\scripts\sanmitra-docs.ps1 -ProjectName "MandirMitra" -DocumentType "ALL"`
2. Use VS Code snippets: `fr-mandirmitra` for temple-specific features
3. Follow cultural accuracy guidelines
